Background Color Change Navigation

This navbar demonstrates smooth background color transitions on hover. Each navigation link changes its background color when you hover over it, creating a modern and interactive feel.

Key Features:

  • Smooth background color transitions
  • Rounded corners for modern design
  • High contrast for accessibility
  • Clean typography and spacing

Color Design Principles:

The color scheme uses complementary colors to create visual hierarchy and ensure good readability. The hover states provide clear feedback to users about interactive elements.

Primary: #3498db
Secondary: #2c3e50
Accent: #e67e22

CSS Transition Properties:

background-color: transparent;
transition: background-color 0.3s ease, color 0.3s ease;
border-radius: 6px;
